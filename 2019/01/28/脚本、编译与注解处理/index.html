Java 平台的脚本

脚本语言是一种通过在运行时解释程序文本，从而避免使用通常的编辑/编译/链接/运行循环的语言。脚本语言有许多优势：

  • 便于快速便跟，鼓励不断试验
  • 可以修改运行着的程序的行为
  • 支持程序用户的定制化
