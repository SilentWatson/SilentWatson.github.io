抽象类和接口的区别

Posted on 2019-02-03 | Post modified: 2019-02-03 | In Java
  • 抽象类是类，Java中类单继承；接口不是类，可以多继承，同时一个类也可以实现多个接口
  • 抽象类中可以有非静态成员变量；接口中的变量必须同时是static和final的变量
  • 抽象类有构造器；接口没有构造器
  • 抽象类中有抽象方法和非抽象方法；接口只有抽象方法（JDK1.8接口可以有实现方法）
